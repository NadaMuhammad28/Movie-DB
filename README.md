# Description

- MovieDBs is a movie database that is provided by an external API [OMDb API](https://www.omdbapi.com)
- **Features:** browsing and searching for movies

---

# Libraries and Tools

- Reactjs
- React-router-dom v6.6.1
- Context API
- Custom hook (UseFetch)
- UseState hook
